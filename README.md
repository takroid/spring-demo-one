 ### JavaのEdition
 
  - Java SE:標準エディション、Javaプログラミングのための基本機能を搭載
  - Java EE:企業用のシステム開発のためのエディション。主にサーバサイド向け
  - java ME:組み込み用。CPUやメモリの消費を抑えるためjava SEより機能が削減されている
